## KANBAN-BOARD

Kanban Board is a tiny task management tool.

### To view the project:

- To build and view the project:
	- `npm i`
	- `npm run view`
	- project will be served on http://localhost:8080/


### To work on the project:

- To start your dev build:
	- `npm i`
	- `npm start`
	- in another tab `npm run watch`
	- make your changes!
